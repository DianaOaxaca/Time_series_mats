1. Generate the metabolic profile¶

The coding regions were searched from the obtained contigs using Prodigal v2.6.3using the -a option, to obtain the translated amino acid sequences of the predicted coding regions and -p meta option. The FASTA files of the resulting sequence contigs have been deposited in the MG-RAST repository under project number mgp80319. The peptide amino acid sequences were then scanned against Pfam-A v30 using hmmsearch. using the -cut_ga -o /dev/null --tblout options. For more details of the domain annotation see Stage 2 in MEBS software manual

Example of the annotated sequences using hmmsearch are found in data/matrix_pfams/ directory

6. Completeness of sulfur cycle using MEBS¶

Due to we have curated manually the sulfur cycle, we known which Pfams are involved in certain metabolic Pathways. Therefore we have generated a file containing the PFAMS and the corresponding KO numbers and the name of the pathway involved, see the tabular file provide in MEBS sofware

Assuming that MEBS is already installed, we compute the completeness of the sulfur cycle in the test_metagenomes directory as an example due to space limits. First we must known the mean size length of the metagenomes.

git clone https://github.com/eead-csic-compbio/metagenome_Pfam_score
cd metagenome_Pfam_score
mebsv1.pl -input test_metagenomes/ -type metagenomic -fdr 0.01

# Computing Mean Size Length (MSL) ...
# 4511045.3_metagenome.faa MSL=32 MSLbin=30
# 4440966.3_metagenome.faa MSL=175 MSLbin=150

Compute the score with a detailed description of the sulfur pathways including kegg visualization using the -size and -keggmap options for the 4440966.3 metagenome

perl scripts/pfam_score.pl  -input test_metagenomes/4440966.3_metagenome.faa.sulfur.hmmsearch.tab  -size 150 -enf cycles/sulfur/entropies.tab -keggmap  Advanced_mode/input_sulfur_data/sulfur_score_kegg_list > completeness.sulfur.score
